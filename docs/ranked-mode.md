# Ranked Mode

Ranked is a local daily programming challenge. It includes:

- daily deterministic challenge selection;
- 180-second timer;
- run examples;
- submit attempt;
- score based on test pass rate and speed bonus;
- RP and local leagues;
- training mode after the ranked attempt;
- local history.

Because this version is frontend-only, Ranked is a personal learning mode, not a real global ranking engine.
